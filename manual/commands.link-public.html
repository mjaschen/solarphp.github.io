8.4. Link Public Assets

./script/solar link-public VendorName

The link-public command creates a combination of directories and symbolic links in the docroot/public/ directory so that public assets from VendorName classes are available to web browsers. (Public linking happens automatically as part of the link-vendor command.)

8.4.1. Example

To link a vendor's public assets into a Solar docroot, issue the following command: 

$ ./script/solar link-public VendorName
Making public symlinks for 'VendorName' ...
    Making public directory docroot/public/VendorName/Controller ... done.
    Making public symlink for VendorName ... done.
    Making public symlink for VendorName ... done.
... done.
$
