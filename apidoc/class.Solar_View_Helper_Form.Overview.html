Overview

Helper for building CSS-based forms.

This is a fluent class; all method calls except fetch() return $this, which means you can chain method calls for easier readability.

Package

This class is part of the Solar_View_Helper_Form package.

Inheritance:

  • Solar_Base

    • Solar_View_Helper

      • Solar_View_Helper_Form

Configuration Keys

  • attribs: Default attributes to use in the <form> tag.

  • request: A Solar_Request dependency injection.

  • descr_part: Where to place descriptions (in the 'label' or the 'value').

  • decorator_tags: Use these decorator tags around form parts.

  • decorator_attribs: Use these attributes for decorator tags.

  • css_classes: Use these CSS classes for form elements.

  • label_suffix: Attach this suffix to all labels.

Constants

None.

Public Properties

The Solar_View_Helper_Form class has no public properties; try the list of all properties.
