Overview

Adapter to fetch roles from an LDAP server.

Package

This class is part of the Solar_Role package.

Inheritance:

  • Solar_Base

    • Solar_Role_Adapter

      • Solar_Role_Adapter_Ldap

Configuration Keys

  • cache: A Solar_Cache dependency injection. Default is to create a Solar_Cache_Adapter_Session object internal to this instance to retain the role list.

  • url: URL to the LDAP server. Takes the format of "ldaps://example.com:389".

  • basedn: The base DN for the LDAP search; example: "o=my company,c=us".

  • filter: An sprintf() filter string for the LDAP search; %s represents the username. Example: "uid=%s".

  • attrib: Use these attributes to find role names.

  • binddn: Bind to the LDAP server as this distinguished name.

  • bindpw: Bind to the LDAP server as with this password.

Constants

None.
