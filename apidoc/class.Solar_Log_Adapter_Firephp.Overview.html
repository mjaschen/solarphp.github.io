Overview

Log adapter for Firephp/Firebug.

Package

This class is part of the Solar_Log package.

Inheritance:

  • Solar_Base

    • Solar_Log_Adapter

      • Solar_Log_Adapter_Firephp

Configuration Keys

  • events: The event types this instance should recognize; a comma-separated string of events, or a sequential array. Default is all events ('*').

  • microtime: Use timestamps with decimal microseconds. Default false.

  • format: The line format for each saved event. Use '%t' for the timestamp, '%e' for the class name, '%e' for the event type, '%m' for the event description, and '%%' for a literal percent. Default is '%t %c %e %m'.

  • output: Output mode. Set to 'html' for HTML, or 'text' for plain text. Default autodetects by SAPI version. Value is ignored by this adapter, since it encodes everything into JSON format.

  • response: A Solar_Http_Response dependency injection.
