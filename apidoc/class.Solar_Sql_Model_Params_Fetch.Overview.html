Overview

A value-object to represent the various parameters for specifying a model fetch() call.

Package

This class is part of the Solar_Sql_Model package.

Inheritance:

  • Solar_Base

    • Solar_Struct

      • Solar_Sql_Model_Params

        • Solar_Sql_Model_Params_Fetch

Constants

None.
