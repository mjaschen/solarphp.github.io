Overview

Applies inflections to words: singular, plural, camel, underscore, etc.

Package

This class is part of the Solar_Inflect package.

Inheritance:

  • Solar_Base

    • Solar_Inflect

Configuration Keys

  • identical: Words that do not change from singular to plural.

  • irregular: Irregular singular-to-plural inflections.

  • to_singular: Rules for preg_replace() to convert plurals to singulars.

  • to_plural: Rules for preg_replace() to convert singulars to plurals.

Constants

None.
