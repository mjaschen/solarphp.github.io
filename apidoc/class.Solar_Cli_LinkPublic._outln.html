Solar_Cli_LinkPublic::_outln()

protected void Solar_Cli_LinkPublic::_outln ( string $text = NULL , mixed $num = 1 , array $replace = NULL )

Prints text to STDOUT and appends a newline.

Inherited from Solar_Controller_Command.

Parameters

  • (string) $text: The text to print to STDOUT, usually a translation key.

  • (mixed) $num: Helps determine whether to get a singular or plural translation.

  • (array) $replace: An array of replacement values for the string.

Returns

  • (void)

Description

Prints text to STDOUT and appends a newline.

If the text is a locale key, that text will be used instead.

Automatically replaces style-format codes for VT100 shell output.
