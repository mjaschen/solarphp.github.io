Configuration

Configuration keys for the Solar_Controller_Console class.

classes

  • (array) Base class names for commands.

  • Default: array ( 0 => 'Solar_Cli', )

routing

  • (array) An array of commands to class names.

  • Default: array ( )

disable

  • (array) An array of command names to disallow.

  • Default: array ( )

default

  • (string) The default command to run.

  • Default: 'help'
