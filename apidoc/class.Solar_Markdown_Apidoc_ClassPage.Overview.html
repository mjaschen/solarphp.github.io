Overview

Replaces class page links in source text with XHTML anchors.

Class page links are in this format ...

[[Class]]
[[Class]]es
[[Class | display this instead]]
[[Class::Page]]
[[Class::$property]]
[[Class::method()]]
[[Class::CONSTANT]]

Package

This class is part of the Solar_Markdown_Apidoc package.

Inheritance:

  • Solar_Base

    • Solar_Markdown_Plugin

      • Solar_Markdown_Apidoc_ClassPage

Configuration Keys

  • markdown: The "parent" Markdown object.

  • constant: A string template for the xml:id for "Constants" page links.

  • overview: A string template for the xml:id for "Overview" page links.

  • method: A string template for the xml:id for individual method page links.

  • other: A string template for the xml:id for all other types of page links.

  • property: A string template for the xml:id for "Properties" page links.

  • package: A string template for the xml:id for "Package" page links.

None.
