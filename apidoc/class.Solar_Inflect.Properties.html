$_Solar_Inflect

(array) Default configuration values.

$_config

(array) Collection point for configuration values.

Inherited from Solar_Base.

Note that you do not define config defaults in $_config directly.

<?php
// DO NOT DO THIS
protected $_config = array(
    'foo' => 'bar',
    'baz' => 'dib',
);

Instead, define config defaults in a protected property named for the class, withan underscore prefix.

For exmple, a "Vendor_Class_Name" class would define the default config array in "$_Vendor_Class_Name". This convention lets child classes inherit parent config keys and values.

<?php
// DO THIS INSTEAD
protected $_Vendor_Class_Name = array(
    'foo' => 'bar',
    'baz' => 'dib',
);

$_identical

(array) A list of words that are the same in singular and plural.

This list is adapted from Ruby on Rails ActiveSupport inflections.

$_irregular

(array) Irregular singular-to-plural conversions.

Array format is "singular" => "plural" and are literal text, not regular expressions.

This list is adapted from Ruby on Rails ActiveSupport inflections.

$_to_plural

(array) Regex rules for converting singular to plural.

Array format is "pattern" => "replacement" for » preg_replace().

All patterns are treated as '/pattern$/i'.

This list is taken from Ruby on Rails ActiveSupport inflections.

$_to_singular

(array) Regex rules for converting plural to singular.

Array format is "pattern" => "replacement" for » preg_replace().

All patterns are treated as '/pattern$/i'.

This list is adapted from Ruby on Rails ActiveSupport inflections.
