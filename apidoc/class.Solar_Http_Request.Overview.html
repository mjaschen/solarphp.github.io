Overview

Factory to return an HTTP request adapter instance.

Package

This class is part of the Solar_Http package.

Inheritance:

  • Solar_Base

    • Solar_Factory

      • Solar_Http_Request

Configuration Keys

  • adapter: The adapter class; for example, 'Solar_Http_Request_Adapter_Stream' (the default). When the curl extension is loaded, the default is 'Solar_Http_Request_Adapter_Curl'.
