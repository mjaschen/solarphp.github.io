Mailing Lists and IRC

IRC Channel

You can join #solar-talk at irc.freenode.net to talk with other Solar users, live! (Except for pmjones, who appears to be a bot.)

Remember to be nice to the others on IRC. :-)

Solar-Talk

The Solar-Talk list is for general discussion about development and use of Solar.

The main mailing list page is here, and you can join the list using the form on that page.

The archives are here.

Remember to be nice when posting to the list. :-)

Solar-Svn

The Solar-Svn list is for commits to the Subversion repository.

The main mailing list page is here, and you can join the using the form on that page.

Solar-Trac

The Solar-Trac list is for changes to the Trac project management system.

The main mailing list page is here, and you can join the using the form on that page.
